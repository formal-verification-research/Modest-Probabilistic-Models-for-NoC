# Previous Models

This directory contains previous works that were likely inspiration for
current works, but are no longer being updated.

- [Boe2023](./Boe2023/): The original modular NoC Modest models are
  contained here.
- [Roberts2020](./Roberts2020/): The original 2x2 monolithic Modest Noc
  model is contained here.
- [Waddoups2024](./Waddoups2024/): The first iteration of the updated
  modular NoC models are contained here.
